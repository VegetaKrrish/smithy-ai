# Smithy AI

Smithy AI is an AI-powered worldbuilding platform designed for storytellers, game masters, writers, and creative teams.

## Features

* AI-generated fantasy worlds
* Rich lore generation
* Geography generation
* Faction creation
* Character generation
* Timeline creation
* Interactive world archive
* Dynamic world exploration interface

## Tech Stack

### Frontend

* React
* TypeScript
* Tailwind CSS
* Vite

### Backend

* Node.js
* Express.js
* MySQL

### AI Integration

* OpenRouter API
* Gemini Models

## Microsoft Hackathon Submission

Smithy AI demonstrates how AI can transform creative ideation into structured and immersive fictional worlds through intelligent content generation and interactive exploration.
